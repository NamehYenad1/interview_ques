
# Card Game

A card game created for technical interview



## Run Locally

Clone the project

```bash
  git clone https://github.com/NamehYenad1/interview_ques.git
```

Go to the project directory

```bash
  cd interview_ques
```

Install dependencies

```bash
  npm install
```

run in console

```bash
  node index.js
```
